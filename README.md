### First Time Setup (or you're restarting because you broken something)

Welcome to my NeoVim setup! Or maybe you've been here before, which is likely too. Here are the steps you must take to get this version of NeoVim fully working!

#### Mac/Linux
- Install NeoVim
```
sudo apt install neovim
```
- Create the `nvim` dir in `.config` and clone this repo inside.
- *NOTE: It is important that you add the `.` at the end*
```
mkdir ~/.config/nvim
cd ~/.config/nvim
git clone <http> === OR === git clone <ssh>
```
- Install ripgrep (It is needed for the grep functionality)
```
sudo apt-get install ripgrep
```

- Install a Nerd Font
    - Go to `nerdfont.com` and find a font.
    - Once you find one, download it and install.
    - Lastly, Google how to set it as your terminal font.
    - Boom.
